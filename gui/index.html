Help

Do screen controls and joystick controls work?

Yes. No setup needed in the app for these. Remember that both take the average position of everyone's movements.

Help! I turned on screen or joystick controls and can't shut down the app!

Press Ctrl + Backspace.

The screen or joystick controls aren't smooth in game!

Each game is built to handle mouse movements differently and this app may not work for all of them.

Viewers can't click with the screen controls!

Nope. This is turned off on purpose. If you give people control of your mouse and also give them the ability to click... bad things can happen. If enough people ask for this it can be incorporated later as a toggle.

Individual vs Global Cooldowns

You can choose to use either individual cooldowns or global cooldowns. Individual means it goes on cooldown for only the person that clicked the button. These are handled in the beam dev lab. Global means it goes on cooldown for everyone. These are handled in this app.

I found a bug! Who should I tell?

Go to this github page and hit the new issues button. Include as much detail as possible.

TIP: Ctrl + Backspace will shut down the program. Useful if you have mouse controls on.
